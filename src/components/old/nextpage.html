Fade In Up

Halo dunia
Code 
/* CSS */
@keyframes fadeInUp { from { opacity: 0; transform: translateY(8px);} to { opacity: 1; transform: translateY(0);} }
.element { animation: fadeInUp .6s ease-out both; }

/* HTML */
<div class="element">Halo dunia</div>
Webflow 
<style>
@keyframes zc-fadeInUp { from { opacity: 0; transform: translateY(8px);} to { opacity: 1; transform: translateY(0);} }
.wf-fade-in-up { animation: zc-fadeInUp .6s ease-out both; }
@media (prefers-reduced-motion: reduce) { .wf-fade-in-up { animation: none !important; } }
</style>

/* Usage (Webflow):
1) Project Settings → Custom Code → Head (paste the <style> block) or use an Embed element.
2) Give your element class: wf-fade-in-up. */

Soft Bounce

Code 
/* CSS */
@keyframes bounceSoft { 0%,100%{ transform: translateY(0);} 50%{ transform: translateY(-12%);} }
.ball { animation: bounceSoft 1.3s cubic-bezier(.22,1,.36,1) infinite; }

/* HTML */
<div class="ball"></div>
Webflow 
<style>
@keyframes zc-bounceSoft { 0%,100%{ transform: translateY(0);} 50%{ transform: translateY(-12%);} }
.wf-bounce-soft { animation: zc-bounceSoft 1.3s cubic-bezier(.22,1,.36,1) infinite; }
@media (prefers-reduced-motion: reduce) { .wf-bounce-soft { animation: none !important; } }
</style>

/* Usage (Webflow):
1) Add an element and give it class: wf-bounce-soft. */

Marquee

Code 
/* CSS */
@keyframes marquee { 0%{ transform: translateX(0);} 100%{ transform: translateX(-50%);} }
.marquee { overflow: hidden; }
.marquee__inner { width: max-content; animation: marquee 10s linear infinite; }

/* HTML */
<div class="marquee">
  <div class="marquee__inner">… items repeated …</div>
</div>
Webflow 
<style>
@keyframes zc-marquee { 0%{ transform: translateX(0);} 100%{ transform: translateX(-50%);} }
.wf-marquee { overflow: hidden; mask-image: linear-gradient(90deg, transparent, #000 10%, #000 90%, transparent); }
.wf-marquee__inner { width: max-content; display: flex; gap: 1.5rem; white-space: nowrap; animation: zc-marquee 10s linear infinite; }
@media (prefers-reduced-motion: reduce) { .wf-marquee__inner { animation: none !important; } }
</style>

/* Usage (Webflow):
1) Wrapper: class wf-marquee
2) Inside: a div with class wf-marquee__inner containing repeated items/text */

Gentle Flip Y

Flip!
Code 
/* CSS */
@keyframes flipY { from{ transform: rotateY(0);} to{ transform: rotateY(180deg);} }
.card { transform-style: preserve-3d; animation: flipY 2.4s ease-in-out infinite alternate; }

/* HTML */
<div class="card">Flip!</div>
Webflow 
<style>
@keyframes zc-flipY { from{ transform: rotateY(0);} to{ transform: rotateY(180deg);} }
.wf-flip-y { transform-style: preserve-3d; animation: zc-flipY 2.4s ease-in-out infinite alternate; }
.wf-flip-card { perspective: 1000px; }
@media (prefers-reduced-motion: reduce) { .wf-flip-y { animation: none !important; } }
</style>

/* Usage (Webflow):
1) Wrapper element: class wf-flip-card
2) Inner element: class wf-flip-y */

Tips

  • Gunakan durasi 300–700ms untuk transisi UI agar terasa halus.
  • Kurangi animasi pada elemen besar; fokuskan pada komponen interaktif.
  • Pertimbangkan prefers-reduced-motion untuk aksesibilitas.
